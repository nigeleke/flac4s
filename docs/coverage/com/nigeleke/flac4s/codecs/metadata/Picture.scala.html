53   //  <32>  The picture type according to the ID3v2 APIC frame:
54   //        0 - Other

62   //        8 - Artist/performer
63   //        9 - Conductor
64   //        10 - Band/Orchestra
65   //        11 - Composer
66   //        12 - Lyricist/text writer
67   //        13 - Recording Location
68   //        14 - During recording
69   //        15 - During performance
70   //        16 - Movie/video screen capture
71   //        17 - A bright coloured fish
72   //        18 - Illustration
73   //        19 - Band/artist logotype
74   //        20 - Publisher/Studio logotype
75   //        Others are reserved and should not be used. There may only be one each of picture type 1 and 2 in a file.
76   //  <32>	The length of the MIME type string in bytes.
77   //  <n*8>	The MIME type string, in printable ASCII characters 0x20-0x7e. The MIME type may also be --> to signify that the data part is a URL of the picture instead of the picture data itself.
78   //  <32>	The length of the description string in bytes.
79   //  <n*8>	The description of the picture, in UTF-8.

83   //  <32>	For indexed-color pictures (e.g. GIF), the number of colors used, or 0 for non-indexed pictures.
